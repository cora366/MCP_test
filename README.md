# Keyboard Input Test

키보드 입력을 시각적으로 확인할 수 있는 웹 기반 테스트 프로그램입니다.

## 기능

- 키를 누르면 화면의 키보드에서 해당 키가 실시간으로 강조 표시
- 누른 키가 상단 입력 표시 영역에 순서대로 기록
- Clear 버튼으로 입력 내역 초기화
- 마지막으로 누른 키 코드 표시

## 실행 방법

`keyboard-test.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 별도 라이브러리 없음)
